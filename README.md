# Kripa

Computer Science Engineering student (B.Tech, 3rd year), building toward machine learning and computer vision — with a preference for projects that ship past the notebook and into something that actually runs.

### Currently building

**AETIS — AI Emergency Traffic Intelligence System**
A traffic-safety system combining three detection pipelines: emergency vehicle detection, road damage detection, and accident detection. Detection models are built on YOLO; the backend runs on FastAPI with PostgreSQL for persistence, containerized with Docker.

**Road Damage Detection**
A standalone computer vision model trained to flag potholes, surface cracks, and broken road sections from image input. Its detection core was later folded into AETIS.

**ML / data work**
Classification, data analysis, and visualization work using pandas, NumPy, and scikit-learn — exercises live in [`AIML`](https://github.com/Kripa0703/AIML), quick experiments prototyped in [`Streamlit`](https://github.com/Kripa0703/Streamlit).
